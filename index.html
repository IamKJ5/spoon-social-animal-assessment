How to Connect to Google Sheets

1. Open your Spreadsheet

Open your Google Spreadsheet and name Columns A to H exactly in row 1:

Row 1 Headers: Timestamp | Full Name | Email | Lion Score | Monkey Score | Dolphin Score | Elephant Score | Primary Style

2. Paste this upgraded Google Apps Script

Inside your sheet, click Extensions > Apps Script, clear the script editor, and paste this robust diagnostic code:

function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    // Form POST (iframe method) always arrives as e.parameter
    var data = e.parameter;

    // 🔒 Domain restriction: only allow @spoonconsulting.com submissions
    var email = (data.email || "").toLowerCase();
    if (!email.endsWith("@spoonconsulting.com")) {
      return ContentService.createTextOutput(
        JSON.stringify({ result: "error", reason: "Unauthorized domain" })
      ).setMimeType(ContentService.MimeType.JSON);
    }
    
    sheet.appendRow([
      new Date(),
      data.name || "Anonymous",
      data.email || "N/A",
      data.lion || 0,
      data.monkey || 0,
      data.dolphin || 0,
      data.elephant || 0,
      data.primary || "Unknown"
    ]);
    return ContentService.createTextOutput(JSON.stringify({"result": "success"})).setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    // If anything breaks, append it directly as a row so facilitators see why!
    try {
      var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
      sheet.appendRow([new Date(), "ERROR DETECTED", error.toString()]);
    } catch(innerErr) {}
    return ContentService.createTextOutput(JSON.stringify({"result": "error", "error": error.toString()})).setMimeType(ContentService.MimeType.JSON);
  }
}

3. Deploy as a Public Web App

  • Click Deploy > New Deployment at the top right.
  • Click the gear icon (Select type) and choose Web App.
  • Set Execute as: Me (your email)
  • Set Who has access: Anyone (CRITICAL STEP)
  • Click Deploy, click Authorize Access, go to Advanced, and approve permissions.
  • Copy the "Web app URL" ending in /exec.

4. Add Link inside this HTML file

Open this HTML file in your code editor. On **Line 330**, replace the URL with your copied link. Save the file and you're good to go!
